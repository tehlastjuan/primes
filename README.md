# primes

Simple program to generate prime numbers.
```
primes - prime number generator

 usage: [amount] [cols]
-----------------------------------------------------------
 [amount] - number of prime numbers to generate
 [cols] - columns to print (default 1)
```

Compile with make
```
$ make
```
Run with ./primes

```
$ ./primes
```
